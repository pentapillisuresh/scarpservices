# Scrap Collection Application

A comprehensive scrap collection management system with Firebase OTP authentication, image upload, and admin dashboard.

## Features

- User registration and authentication with JWT (3 years validity)
- Firebase phone OTP verification
- Scrap category management
- Collection request creation with multiple images
- Address management with geolocation
- Admin dashboard with request management
- Image upload with local and Cloudinary storage
- Real-time notifications

## Prerequisites

- Node.js 16+
- MySQL 8+
- Firebase project for OTP
- Cloudinary account (for production image storage)
